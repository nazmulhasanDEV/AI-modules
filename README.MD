## ðŸ”¹ 2. Manage Conda Environments

### Check existing environments

```bash
conda env list
```

### Create new environment (example: NLP course)

```bash
conda create -n nlp-env python=3.11
```

### Activate environment

```bash
conda activate nlp-env
```

### Deactivate environment

```bash
conda deactivate
```

### Clone (copy) environment

```bash
conda create --name new-env --clone old-env
```

### Remove environment

```bash
conda remove --name old-env --all
```

### Open Project

* `File â†’ Open Folder` â†’ choose your project folder (e.g., `AI-modules`).
